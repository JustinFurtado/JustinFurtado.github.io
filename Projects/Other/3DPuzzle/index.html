3D Puzzle Game

This was a very simple assignment - an open-ended final for a web development course that required us to make something in a browser using a library we had never used before. Because of this I took the opportunity to learn Three.js, not in-depth, but enough to make the simple above puzzle game.

The puzzle can be solved in a minimum of 28 moves, based on my personal tests. To solve it, get every square to go from its 'starting color' to its 'alternate color' If you want to start over, you can refresh the page.
