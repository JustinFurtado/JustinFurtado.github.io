Extended Details

For the final for the Mobile Application Development course, I managed to convince my not-so-game-enthusiastic group members to make a game with me for the open-ended final assignment

We made a simple game using OpenGL to render objects to the device. Art assets were made by Quintyn, and code was written by myself and Joshua.

The game mechanic is simple, tap the targets, if you miss three you get a game over, if you tap an 'x' object, you lose instantly. Objects spawn faster and faster as the game goes on, and you compete for local high scores saved to the device.

It was developed using an android emulator, but did work on a device when we installed it directly after turning on developer mode and what not.
